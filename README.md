# TalentHub - Recruitment Dashboard

TalentHub adalah aplikasi dashboard manajemen rekrutmen modern yang dibangun menggunakan React 19, Vite 7, TypeScript, Tailwind CSS v4, dan terintegrasi dengan Supabase sebagai Backend-as-a-Service (BaaS).

Proyek ini dibuat sebagai bagian dari pemenuhan Technical Assessment / Project Recruitment.

## 🚀 Fitur Utama
- **Dashboard Analytics:** Visualisasi data rekrutmen menggunakan grafik interaktif dari Recharts.
- **Candidate Management:** Pelacakan status pelamar (Applied, Interview, Technical Test, Offered, Rejected) secara *real-time*.
- **Export Data:** Fitur untuk mengunduh laporan data kandidat langsung ke dalam format Excel (.xlsx).
- **Responsive Design:** Tampilan antarmuka yang optimal dan bersih di perangkat desktop maupun mobile menggunakan Tailwind CSS v4.

## 🛠️ Tech Stack
- **Frontend:** React 19, TypeScript, Vite 7
- **Styling:** Tailwind CSS v4 (menggunakan `@tailwindcss/vite` compiler baru)
- **Database & State:** SupabaseJS Client
- **Icons & Charts:** Lucide React, Recharts
- **Data Processor:** SheetJS (`xlsx`)
